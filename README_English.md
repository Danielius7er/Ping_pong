# Pong Game 🎮

A classic Pong game implementation using HTML5, CSS3, and vanilla JavaScript with real-time AI opponent.

## Features ✨

- **Player vs Computer Gameplay**: Control the left paddle to compete against an intelligent AI opponent
- **Dual Control Methods**: 
  - Mouse Y-position tracking for smooth paddle control
  - Arrow Keys (↑ Up / ↓ Down) for alternative control
- **Advanced Physics Engine**:
  - Realistic ball bouncing with wall collision detection
  - Paddle collision with spin mechanics
  - Ball speed progression
- **Intelligent AI Opponent**: Computer paddle with medium difficulty and reaction delays
- **Real-time Scoreboard**: Live score tracking for both player and computer
- **Visual Effects**:
  - Neon-themed UI with glow effects
  - Smooth animations and transitions
  - Color-coded paddles and ball
- **Game Controls**:
  - Start/Stop game functionality
  - Reset score button
  - Pause by stopping the game

## How to Play 🕹️

1. Open `index.html` in your web browser
2. Click the **"Start Game"** button to begin
3. Control your left paddle using:
   - **Mouse**: Move your cursor up/down
   - **Arrow Keys**: Press ↑ (Up) or ↓ (Down)
4. Try to hit the ball past the computer's paddle to score points
5. First to reach a target score wins! (or just have fun!)

## Game Rules 📋

- The left paddle is controlled by the player
- The right paddle is controlled by the AI computer
- The ball bounces off the top and bottom walls
- Scoring a point happens when the opponent fails to hit the ball
- The ball's speed increases slightly with each paddle hit
- Ball spin is added based on where it hits the paddle
